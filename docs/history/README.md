# 历史文档

这里存放项目历史记录与长周期开发日志。

## 目录职责

- `agent_handoff.md`
  当前状态、当前优先级、当前验证要求与接手约束
- `development_logs/`
  按轮次追加的开发日志；每个文件最多 `500` 行
- `project_scan_2026-03-21.md`
  历史代码扫描记录
- `balance_notes.md`
  早期平衡测试记录
- `historical_event_review.md`
  历史事件审阅与问答摘录
- `ui_prototype.html`
  历史 UI 原型

## 维护规则

- 多轮开发历史不要再写回 `development_plan` 或 `agent_handoff`
- 当前开发日志接近 `500` 行时，新建下一份 `development_log_00N.md`
- `agent_handoff` 只保留当前态；逐轮变化统一写入 `development_logs/`
